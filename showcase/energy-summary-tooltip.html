Energy Summary Tooltip

Interactive Demo

Hover over the info button to see the premium energy summary tooltip with real-time dashboard data.

ℹ️ Informacoes de Energia Online

Features

RFC-0105: Premium tooltip component showing comprehensive energy dashboard summary.

📊 Total Device Count

Shows the total number of monitored energy devices across all categories.

🏢 Category Breakdown

Hierarchical tree view with device counts and consumption per category (Entrada, Lojas, Area Comum).

🏭 Subcategories

Area Comum expands to show Climatizacao, Elevadores, Esc. Rolantes, Chillers, Fancoils, Bombas, etc.

🟢 Status Matrix

Visual breakdown of device statuses: Normal, Alert, Failure, Standby, Offline, No Consumption.

⚡ Total Consumption

Aggregated consumption total displayed prominently in the footer with proper unit formatting.

🔄 Real-Time Sync

Data synchronized with TELEMETRY and MAIN_VIEW widget controllers for live updates.

📌 PIN Button

Fix the tooltip in place so it stays visible even when mouse leaves. Click again to unpin.

▣ Maximize/Restore

Expand the tooltip to full screen for easier reading. Click again to restore original size.

✍ Drag to Move

Click and drag the header to move the tooltip anywhere on screen. Maintains position until closed.

✖ Close Button

Instantly close the tooltip with a single click. Also resets all states (pin, maximize, position).

Usage

How to integrate the Energy Summary Tooltip in your widgets.

// Import the component import { EnergySummaryTooltip } from '@myio/js-library'; // Build summary data from widget state const getSummaryData = () => { return EnergySummaryTooltip.buildSummaryFromState( { entrada: STATE.entrada, consumidores: STATE.consumidores, grandTotal: STATE.grandTotal }, RECEIVED_DATA ); }; // Attach tooltip to trigger element const triggerElement = document.getElementById('btnInfoSummary'); const cleanup = EnergySummaryTooltip.attach(triggerElement, getSummaryData); // Later: cleanup() to remove event listeners

Data Structure

The tooltip expects data in the following format.

interface DashboardEnergySummary { totalDevices: number; totalConsumption: number; unit: 'kWh' | 'MWh'; byCategory: CategorySummary[]; byStatus: { normal: number; alert: number; failure: number; standby: number; offline: number; noConsumption: number; }; lastUpdated: string; // ISO 8601 } interface CategorySummary { id: string; name: string; icon: string; deviceCount: number; consumption: number; percentage: number; children?: CategorySummary[]; }
