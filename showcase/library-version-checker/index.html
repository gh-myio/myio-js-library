RFC-0137: LibraryVersionChecker Component

Componente que exibe a versao da biblioteca e verifica atualizacoes no npm com tooltip premium

Demonstracao ao Vivo

O componente verifica automaticamente a versao mais recente no npm e exibe um indicador visual. Clique no icone de status para ver o tooltip detalhado com instrucoes de atualizacao.

Como Usar

O componente createLibraryVersionChecker injeta todo o conteudo HTML e CSS necessario no container fornecido. Basta passar o elemento container e as opcoes de configuracao.

// No ThingsBoard widget (MENU/controller.js) const container = document.getElementById('lib-version-display'); if (MyIOLibrary.createLibraryVersionChecker) { MyIOLibrary.createLibraryVersionChecker(container, { packageName: 'myio-js-library', currentVersion: MyIOLibrary.version || 'unknown', onStatusChange: (status, current, latest) => { console.log(`Status: ${status}`); }, }); }
