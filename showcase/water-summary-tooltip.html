Water Summary Tooltip

Interactive Demo

Hover over the info button to see the premium water summary tooltip with dashboard data.

💧 Informacoes de Agua Online

Features

RFC-0105: Premium tooltip component showing comprehensive water dashboard summary.

📊 Total de Medidores

Shows the total number of water meters across all categories.

💧 Unidade m3

All consumption values displayed in cubic meters (m3) with proper Brazilian formatting.

🏢 Category Breakdown

Water-specific categories: Entrada, Lojas, Banheiros (optional), Area Comum, Pontos Nao Mapeados.

🚿 Banheiros Toggle

Optional category for bathrooms - controlled by includeBathrooms setting in widget config.

🟢 Status Matrix

Visual breakdown of meter statuses: Normal, Alert, Failure, Standby, Offline, No Consumption.

📌 PIN Button

Fix the tooltip in place so it stays visible even when mouse leaves. Click again to unpin.

▣ Maximize/Restore

Expand the tooltip to full screen for easier reading. Click again to restore original size.

✍ Drag to Move

Click and drag the header to move the tooltip anywhere on screen.

Usage

How to integrate the Water Summary Tooltip in your widgets.

// Import the component import { WaterSummaryTooltip } from '@myio/js-library'; // Build summary data from widget state const getSummaryData = () => { return WaterSummaryTooltip.buildSummaryFromState( STATE_WATER, RECEIVED_DATA, STATE_WATER.includeBathrooms ); }; // Attach tooltip to trigger element const triggerElement = document.getElementById('btnInfoSummary'); const cleanup = WaterSummaryTooltip.attach(triggerElement, getSummaryData); // Later: cleanup() to remove event listeners

Data Structure

The tooltip expects data in the following format.

interface DashboardWaterSummary { totalDevices: number; totalConsumption: number; unit: 'm3'; byCategory: WaterCategorySummary[]; byStatus: { normal: number; alert: number; failure: number; standby: number; offline: number; noConsumption: number; }; lastUpdated: string; // ISO 8601 includeBathrooms?: boolean; } interface WaterCategorySummary { id: string; // 'entrada' | 'lojas' | 'banheiros' | 'areaComum' | 'pontosNaoMapeados' name: string; icon: string; deviceCount: number; consumption: number; percentage: number; }
